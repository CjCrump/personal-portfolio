# Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript to showcase projects, skills, and provide a functional contact form using Netlify Forms.

**Live Site:** https://chanceit.netlify.app  
**Repository:** https://github.com/yourusername/your-repo-name

---

## Overview

This project serves as my primary personal portfolio and landing page. It highlights my technical skills, selected projects, and provides a direct way to contact me through a serverless form integration.

The site is designed to be:
- Mobile-first
- Fast and lightweight
- Easy to maintain and extend
- Deployed using modern static hosting practices

---

## Features

- Responsive layout built with **Flexbox**
- Clean, semantic HTML structure
- Project showcase section with live demos
- Smooth scroll navigation
- Fully functional contact form using **Netlify Forms**
- Client-side form validation
- Spam protection via honeypot field
- Success and error feedback without page reload

---

## Tech Stack

- **HTML5** – semantic structure and accessibility
- **CSS3** – Flexbox layout, responsive design, custom styling
- **JavaScript (Vanilla)** – DOM manipulation, validation, UX enhancements
- **Netlify** – hosting and serverless form handling
